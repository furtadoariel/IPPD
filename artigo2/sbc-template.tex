\section{Introdução}\label{sec:intro}
O projeto do \textit{Apache Hadoop} desenvolve um \textit{software} para computação confiável, escalonável e distribuída. A biblioteca dessa ferramenta é um \textit{framework} que permite processamento distribuído de grandes quantidades de dados através de \textit{clusters} de computação usando modelos simples de programação. Ele é projetado para escalonar a partir de um único servidor para milhares de máquinas , cada um oferecendo computação e armazenamento local.\\
Esse artigo tem como objetivo mostrar como foi feito a instalação passo a passo dessa ferramenta em uma máquina, bem como seus requisitos, e foi projetado da seguinte forma: na Sessão \ref{sec:intro} foi feito uma introdução ao trabalho, na Sessão \ref{sec:met} foi mostrado a Metodologia usada no trabalho, na Sessão \ref{sec:inst} foi mostrado o passo a passo da instalação da ferramenta.

\subsection{Instalação do \textit{Hadoop}}\label{sub:instHadoop}
\subsubsection{\textit{Download, extração e movimentação do pacote}}
\begin{itemize}

 \item \textit{wget http://mirrors.sonic.net/apache/hadoop/common/hadoop-2.6.0/hadoop-2.6.0.tar.gz}\footnote[1]{Download do Pacote}
 \item tar xvzf hadoop-2.6.0.tar.gz\footnote[2]{Extração do pacote}
 \item sudo mv * /usr/local/hadoop \footnote[3]{Substitua * pelo nome da pasta do Hadoop}
 \item sudo chown -R user:group /usr/local/hadoop \footnote[4]{substitua user e group pelos respectivos usuario e grupo}
 
\end{itemize}
